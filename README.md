KMY Browser
Lightweight Native Browser for Windows (64-bit)

KMY Browser is a fast, minimal web browser built for Windows using
native system technologies. It focuses on performance, simplicity,
and a clean browsing experience.

FEATURES
- Native Windows application
- Fast startup
- Lightweight (no Electron)
- Brave Search as default search engine
- Smart address bar (.com auto-detection)
- Built-in About dashboard
- No ads
- No tracking

SYSTEM REQUIREMENTS
- Windows 10 or Windows 11 (64-bit)
- Microsoft Edge WebView2 Runtime
  (usually preinstalled on modern Windows)

If the browser does not open, install WebView2 Runtime:
https://developer.microsoft.com/microsoft-edge/webview2/
